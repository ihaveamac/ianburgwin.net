What is it I do with 3DS homebrew?

Sometimes I'd make videos trying out a new piece of homebrew and want to show it off.

Other times (and most of the time, really) I just like to break things by using it improperly. Or try out something that's really broken.

I have five 3DS systems, so I'm more willing to try more dangerous things, such as formatting a system through HANS (which I have done before!). Here's a link to them, including DS!

  • YouTube

I occasionally do testing for homebrew as well. I've helped Rinnegatamante a few times running homebrew on ninjhax 2.x, as well as dumping and testing for CHMM2's theme shuffle support. I enjoy doing it so I keep going.

Here's one of my "completed" projects, and you've probably already heard of it already: the updater that is included with mashers's Grid Launcher. It was annoying and interesting with the workarounds I had to use; for example, this site needs to cache the latest version as opposed to the 3DS connecting to GitHub directly. There are more details in the repository if you are interested.

  • GitHub (mashers-gl-updater)

This is my second homebrew: screenshot-merge. It simply takes screenshots taken with ninjhax 2.x or NTR CFW and merges the top and bottom screens into a single image, without using a computer.

  • GitHub (screenshot-merge)

I was working on a file manager of sorts. At this point I should just rewrite it.

  • GitHub (another-file-manager)

I created a downloader for game covers for GameTDB's 3DS section. The covers are used with mashers's Grid Launcher.

  • GitHub (3DSFlow-downloader)

I started working on a repository for *hax 2.7, which is going to support titles encrypted with a new kind of crypto starting with 9.6.0-24.

  • GitHub (9.6-dbgen-xmls)
